fypy使用说明
==============================

    随着团队成员日益扩大，涉及的方向、领域也扩展了很多，
    当前可提供 *定制化* 的功能需求（地物分类、卫星产品定量、定性反演等）

    - 大气
    - 遥感

    - 海洋
    - 生态
    - 灾害
    - 农业
    - ...

 :计算机语言:
    - python
    - c/c++
    - Fortran
    - IDL
    - matlab
    - R
    - ...

 :fypy包下载地址: https://gofile.me/7ieAe/gD3YpGdhF

.. note::
    由于pypi强制要求F2A验证，导致包的发布受阻，当前提供云下载后进行安装：
    下载地址：https://gofile.me/7ieAe/gD3YpGdhF
    （ `fypy包 <https://gofile.me/7ieAe/gD3YpGdhF>`_ ）

    遥感数据处理、气象计算公式、
    栅格矢量基础操作等功能。

    未来计划：

    - 风云卫星产品数据的处理、图像绘制等功能
    - 气象行业常用的计算公式
    - 常用的散点图、直方图、时序图绘制等

.. warning::

    当前 ``fypy`` 库还不是一个稳定的版本，正在对接口API
    进行测试补充，争取把接口满足大家需求。



``fypy`` 是一个支持 ``python`` 的库，并支持以下功能：


+------------------------+----------------------+--------------------------------+----------+
|         库名           |    对象              |         功能                   |   备注   |
+========================+======================+================================+==========+
| fypy                   | fy3                  | 极轨卫星FY3的5分钟块、轨道数据 |          |
|                        |                      | 的投影、拼接裁剪               |          |
|                        +----------------------+--------------------------------+----------+
|                        | fy4                  | 卫星数据的经纬度转行列号，     |          |
|                        |                      | 行列号转经纬度，以及对数据进行 |          |
|                        |                      | 投影转换、裁剪等功能（待发布） |          |
|                        +----------------------+--------------------------------+----------+
|                        | h8                   |                                |          |
|                        +----------------------+--------------------------------+----------+
|                        | modis                |                                |          |
+------------------------+----------------------+--------------------------------+----------+
