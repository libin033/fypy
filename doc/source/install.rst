========================
Install
========================

.. tip::
    由于pypi强制要求F2A验证，导致包的发布受阻，当前提供云下载后进行安装：
    下载地址：https://gofile.me/7ieAe/gD3YpGdhF
    （ `fypy包 <https://gofile.me/7ieAe/gD3YpGdhF>`_ ）

pip安装fypy
--------------------------

.. code-block:: bash

   python -m pip install fypy
   python -m pip install --upgrade fypy


conda安装fypy
--------------------------

.. code-block:: bash

   conda install fypy

.. warning::

    即将发布conda的fypy包
